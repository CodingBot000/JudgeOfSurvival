# JudgeOfSurvival

React + Vite web port of **The Judge of Survival**, a rule-driven survival judgement simulation originally implemented in Godot.

## Features

- SPA screens for trial, roster, log, and judgement.
- Korean default UI with English language toggle.
- Minor and major god power system.
- Lifeboat turn progression, event rules, death checks, end conditions, and final judgement.
- Local-only gameplay with no API or network dependency.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run smoke
npm run build
```
